Many2many through a view


Many2many through a view is technical addon developed by Jorge Soto Garcia.

This addon can be used to create a many to many relationship between two models using an SQL view as a middle relation.

At this time this addon is in alpha stage, thus use it on your own risk.

Usage

  1. Add m2m_through_view as dependency to your addon
  2. Import field class from odoo.addons.m2m_through_view.field import Many2manyThroughView
  3. Declare field my_field = Many2manyThroughView('My field', sql='...')

Known Bugs

  • Many2manyThroughView fields are not supported in xmlrpc

Also it is possible to use this field as: my_field = odoo.fields.BigInt()
